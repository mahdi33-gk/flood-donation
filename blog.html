Important FAQ

What is the document object model(DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a webpage as a tree of objects, allowing JavaScript to manipulate HTML elements, styles, and content dynamically.

How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById('myId') for an ID, document.querySelector('.myClass') for a class, or document.querySelectorAll('div') for all

elements.

What is event delegation in the context of DOM, and why it is useful?

Event delegation is a technique that allows a single event listener to be added to a parent element, which handles events for its child elements. This is useful for improving performance, managing dynamically added elements, and simplifying code by reducing the number of event listeners needed.

How do you manipulate an element's attribute's and styles using the DOM ?

You can manipulate an element's attributes and styles in the DOM using methods like element.setAttribute('attrName', 'value') to set an attribute, and element.style.propertyName = 'value' to change a style. For example, element.classList.add('className') adds a CSS class.
